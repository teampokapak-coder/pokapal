# Blog Post Generation Prompt

**Copy this prompt for Claude:**

---

Create a blog post for PokaPal (Pokemon TCG master set collection app). Provide:

1. **Title**: Catchy, SEO-friendly (50-60 chars)
2. **Slug**: URL-friendly version (lowercase, hyphens)
3. **Excerpt**: 150-200 character summary
4. **Content**: 3-5 paragraphs in HTML format (use <p>, <h2>, <h3>, <ul>, <li> tags)
5. **Keywords**: 5-8 comma-separated (e.g., "pokemon tcg, collecting, master set")
6. **Meta Title**: Optional SEO title (60 chars max)
7. **Meta Description**: Optional SEO description (150-160 chars)
8. **Linked Content**: Optional set name or Pokemon # to link

Format:
- Title: [title]
- Slug: [slug]
- Excerpt: [excerpt]
- Keywords: [keywords]
- Content: [HTML content]
- Meta Title: [optional]
- Meta Description: [optional]
- Linked Content: [set or Pokemon #]

---

**Topic suggestions:** Base Set guide, rarest cards, starting master sets, card rarities, collection organization
